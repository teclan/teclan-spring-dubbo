# SpringBoot+Dubbo

## 安装 zookeeper

[点此下载](https://mirrors.tuna.tsinghua.edu.cn/apache/zookeeper/)`zookeeper`
 
## 启动 zookeeper

解压进入 `bin` 目录，以windwos为例，执行

```
cd path/to/zookeeper/bin
zkServer
```

## 使用 zookeeper 客户端

```
cd path/to/zookeeper/bin
zkCli
```

## 运行程序

程序启动后，访问 `http://localhost:8081/hello?name=xxx`,详情请看源码
